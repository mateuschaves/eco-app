## Technologies

:white_check_mark: React Native\
:white_check_mark: Styled components\
:white_check_mark: React Navigation

## Running the app

```bash

Install dependencies

$ npm install
or
$ yarn install


For Android 

$ npm run android
or
$ yarn android

For IOS

$ npm run ios
or
$ yarn ios
```
